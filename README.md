# ONTechUConnect

## Clone Repository
### Enter directory on computer that you would like the application
```
git clone git@github.com:ethanelliott/ONTechUConnect.git
```
## Start Server
### Enter "server" directory
```
yarn install
```
```
yarn start
```

## Start Front End
### Enter "ontechu-connect" directory
```
yarn install
```
```
yarn serve
```
